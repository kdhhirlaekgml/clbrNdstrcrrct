# clbrNdstrcrrct

## Camera Calibration
영상 주소 넣기
스페이스바를 눌러 이미지 선택
엔터키를 눌러 이미지 확정

### Calibration Parameters

- **선택된 이미지 수:** 5개

- **초점 거리:**
  - `fx`: 654.60440271 px
  - `fy`: 687.54028032 px

- **주점 좌표:**
  - `cx`: 547.56880934 px
  - `cy`: 598.78802921 px

- **RMSE:** 0.819048498620728 px

### Distortion Correction
camera_calibration.py에서 나온 결괏값(K, dist_coeff) 입력
